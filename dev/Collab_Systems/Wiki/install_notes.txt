--- +TWiki-4.1.1 -------------------
If you're using Apache, run it in non-chrooted mode, or create a correct
environment with sendmail(8), egrep(1) etc. You should add
"Include /var/www/conf/httpd-twiki.conf" to your httpd.conf.

You have to customize your twiki installation (like WIKIWEBMASTER)
in /var/www/htdocs/twiki/data/TWiki/TWikiPreferences.txt

Pages:
* http://localhost/twiki/
* http://localhost/twiki/bin/view/TWiki
* http://localhost/twiki/bin/view/Main/WebHome

To start the httpd daemon in a non chroot use the -u flag as stated in the man page -- Michael

perform the configure as stated in the documentation

Searching:

Twiki expects "egrep" and "fgrep" to use the "-H" option which is popular with modern versions.  On Open BSD this feature is not enabled and must be removed in the configuration.  This works as expected.
