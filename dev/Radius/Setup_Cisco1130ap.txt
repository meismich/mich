CLI commands to setup bare/factory default WAP

To be done manually:

Setup Administrator:
	service password-encryption
	enable secret 0 <is a!>
	username issadmin password 0 <clueless>
	no username Cisco

Setup IP Address/Hostname:
	int bvi1 ip address x.x.x.x 255.255.0.0
	hostname campus_dept_no
	ip default-gateway x.x.y.254



Setup AAA:
	aaa new-model

	aaa group server radius rad_eap
		server 10.32.100.78 auth-port 1812 acct-port 1813
	exit

	aaa authentication login eap_methods group rad_eap
	no aaa authentication login mac_methods local
	aaa authentication exec default local

Setup Radius:
	ip radius source-interface BVI1
	radius-server retransmit 3
	radius-server attribute 32 include-in-access-req format %h
[Manual]
	radius-server host 10.32.100.78 auth-port 1812 acct-port 1813 key 0 <clueless>

Setup Wireless SSID:
	dot11 ssid aptest
		authentication open eap eap_methods
		authentication network-eap eap_methods
		authentication key-management wpa
		guest-mode
	exit

Setup Radios:
	int Dot11Radio 0
		no shutdown
		encryption mode ciphers tkip
		ssid aptest
	exit
	int Dot11Radio 1
		no shutdown
		encryption mode ciphers tkip
		ssid aptest
	exit

Setup SNMP & NTP:
	snmp-server contact "issadmin@apeagers.com.au"
	snmp-server location "CAMPUS - Brand Department - Number"
	snmp-server community apsnmpro ro
	snmp-server community apsnmprw rw
	sntp server 10.1.1.43
